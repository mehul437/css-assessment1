What is CSS?

  1. CSS stands for Cascading Style Sheet.

  2. CSS describes how HTML elements to be displayed on screen,paper,or in other media.

  3. CSS is not neccessay CSS saves a lot of work.It can control the layout of multiple web pages all at once.

  4. External Stylesheet are stored in CSS files.

CSS can be added to HTML document in 3 ways:

  • Inline - by using the style attribute inside HTML elements
  • Internal - bys using a style element in head section
  • External - by suing a link element to link to an External file
  • The most common way to add CSS ,to keep style in External CSS file.however in this tutorial we will use inline and internal styling, because it is easier to demonstrate and easier for you to try it yourself.
